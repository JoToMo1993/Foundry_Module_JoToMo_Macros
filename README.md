# Foundry JoToMo Macros

This module adds a compendium to your Foundry, containing a collection of macros.
The module is mainly to better manage the updates of those macros as well as managing which ones are rolled out.

## Setup / Installation

To install this module, you have to add the module.json's address into the module-manager of foundry.
The url is https://raw.githubusercontent.com/JoToMo1993/Foundry_Module_JoToMo_Macros/refs/heads/main/module.json
